A fact. Any piece of information that you can think of. Don't worry about grammar or language, we'll cover that for you.

FactBase Powering the exchange of knowledge in a simple manner.

FactBase is a personal project designed by Robert Casanova, a student at the University of Florida.
It utilizes SQL to store parsed versions of entered strings in a database. It then fetches a string that
another user entered into the database and presents it.
